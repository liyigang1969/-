## 🏠 家庭基本概念

### 家庭定义
1. **生物学定义**: 基于血缘、婚姻或收养关系组成的社会单位
2. **社会学定义**: 共同生活、经济合作、情感支持的基本社会群体
3. **法律定义**: 具有权利义务关系的亲属团体

### 家庭功能
1. **生育功能**: 繁衍后代
2. **经济功能**: 生产、消费、财产继承
3. **教育功能**: 社会化过程，价值观传承
4. **情感功能**: 情感支持、安全感
5. **保护功能**: 照顾老弱病残

## 🧬 血缘关系体系

### 直系血亲
**定义**: 有直接血缘关系的亲属

#### 向上追溯（尊亲属）：
1. **父母**:
   - 父亲、母亲
   - 生物学父母、养父母

2. **祖父母**:
   - 祖父（爷爷）、祖母（奶奶） - 父亲的父母
   - 外祖父（外公）、外祖母（外婆） - 母亲的父母

3. **曾祖父母**:
   - 曾祖父、曾祖母
   - 外曾祖父、外曾祖母

4. **高祖父母**:
   - 高祖父、高祖母
   - 以此类推：太高祖、远祖、鼻祖

#### 向下追溯（卑亲属）：
1. **子女**:
   - 儿子、女儿
   - 生物学子女、养子女

2. **孙辈**:
   - 孙子、孙女 - 儿子的子女
   - 外孙、外孙女 - 女儿的子女

3. **曾孙辈**:
   - 曾孙、曾孙女
   - 外曾孙、外曾孙女

4. **玄孙辈**:
   - 玄孙、玄孙女
   - 以此类推：来孙、晜孙、仍孙、云孙、耳孙

### 旁系血亲
**定义**: 有间接血缘关系的亲属

#### 兄弟姐妹关系：
1. **全血缘兄弟姐妹**:
   - 同父同母
   - 称呼：哥哥、弟弟、姐姐、妹妹

2. **半血缘兄弟姐妹**:
   - 同父异母或同母异父
   - 称呼相同，但血缘关系不同

3. **堂兄弟姐妹**:
   - 父亲的兄弟的子女
   - 堂哥、堂弟、堂姐、堂妹

4. **表兄弟姐妹**:
   - 父亲的姐妹的子女 → 姑表
   - 母亲的兄弟的子女 → 舅表
   - 母亲的姐妹的子女 → 姨表
   - 表哥、表弟、表姐、表妹

#### 叔伯姑舅姨关系：
1. **叔伯**:
   - 父亲的哥哥 → 伯父（伯伯）
   - 父亲的弟弟 → 叔父（叔叔）

2. **姑**:
   - 父亲的姐妹 → 姑母（姑姑）

3. **舅**:
   - 母亲的兄弟 → 舅父（舅舅）

4. **姨**:
   - 母亲的姐妹 → 姨母（姨妈）

#### 侄甥关系：
1. **侄子、侄女**:
   - 兄弟的子女
   - 称呼：侄子、侄女

2. **外甥、外甥女**:
   - 姐妹的子女
   - 称呼：外甥、外甥女

### 姻亲关系
**定义**: 通过婚姻建立的亲属关系

#### 配偶关系：
1. **丈夫、妻子**
2. **前夫、前妻**（离婚后）
3. **亡夫、亡妻**（配偶去世后）

#### 配偶的血亲：
1. **公婆**:
   - 丈夫的父母 → 公公、婆婆

2. **岳父母**:
   - 妻子的父母 → 岳父、岳母

3. **妯娌**:
   - 兄弟的妻子之间的关系

4. **连襟**:
   - 姐妹的丈夫之间的关系

#### 血亲的配偶：
1. **儿媳**:
   - 儿子的妻子

2. **女婿**:
   - 女儿的丈夫

3. **嫂子、弟媳**:
   - 哥哥的妻子、弟弟的妻子

4. **姐夫、妹夫**:
   - 姐姐的丈夫、妹妹的丈夫

## 🏛️ 中国传统亲属关系体系

### 九族五服制度

#### 九族范围：
1. **高祖辈** → **曾祖辈** → **祖辈** → **父辈** → **己身**
2. **子辈** → **孙辈** → **曾孙辈** → **玄孙辈**

#### 五服制度（丧服制度）：
1. **斩衰**（三年）:
   - 子女为父母
   - 妻为夫
   - 臣为君

2. **齐衰**（三年、一年、五月、三月）:
   - 孙为祖父母
   - 夫为妻
   - 为曾祖父母等

3. **大功**（九月）:
   - 为堂兄弟姐妹
   - 为姑、姐妹、女儿等

4. **小功**（五月）:
   - 为从祖父母
   - 为从祖兄弟姐妹

5. **缌麻**（三月）:
   - 为族曾祖父母
   - 为族兄弟姐妹等

### 宗法制度
1. **大宗小宗**:
   - 大宗：嫡长子系统，继承宗庙
   - 小宗：庶子系统，五世而迁

2. **昭穆制度**:
   - 左昭右穆，区分辈分
   - 宗庙中祖先牌位排列方式

3. **嫡庶之分**:
   - 嫡子：正妻所生
   - 庶子：妾室所生
   - 继承权差异

## 🌍 不同文化的亲属关系

### 西方亲属关系特点
1. **简单化**:
   - 不区分堂表亲（统称cousin）
   - 不区分叔伯舅（统称uncle）
   - 不区分姑姨（统称aunt）

2. **核心家庭**:
   - 强调夫妻和未成年子女的核心家庭
   - 与扩展家庭关系相对松散

3. **个人主义**:
   - 强调个人独立性
   - 亲属关系责任相对较轻

### 中国亲属关系特点
1. **复杂化**:
   - 严格区分父系母系
   - 严格区分长幼次序
   - 丰富细致的称呼体系

2. **家族观念**:
   - 强调家族整体利益
   - 亲属网络广泛紧密
   - 家族责任和义务重要

3. **祖先崇拜**:
   - 重视祖先祭祀
   - 家谱族谱记录
   - 家族传承意识强

### 其他文化特点
1. **母系社会**:
   - 如摩梭人，血缘按母系计算
   - 舅舅承担父亲角色

2. **双系社会**:
   - 同时重视父系和母系
   - 如一些东南亚文化

3. **氏族制度**:
   - 如苏格兰氏族
   - 共同祖先，特定图腾

## 📜 亲属称谓体系

### 汉语亲属称谓特点
1. **区分细致**:
   - 区分父系母系（叔 vs 舅，姑 vs 姨）
   - 区分长幼次序（伯 vs 叔，哥 vs 弟）
   - 区分血缘姻亲（侄 vs 甥，媳 vs 婿）

2. **尊称敬语**:
   - 对长辈用尊称（您、老人家）
   - 对平辈用礼貌称呼
   - 对晚辈用爱称

3. **排行称呼**:
   - 大哥、二哥、三哥
   - 大姐、二姐、三姐
   - 大舅、二舅、小舅

### 称呼的社会功能
1. **身份确认**: 明确在家族中的位置
