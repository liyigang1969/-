## 一、文艺复兴与大航海时代

### 1.1 文艺复兴（14-17世纪）
#### 意大利文艺复兴
1. **人文主义兴起**:
   - 彼得拉克"人文主义之父"
   - 研究古典文献，强调人的价值
   - 世俗化倾向

2. **艺术成就**:
   - **达·芬奇**:《蒙娜丽莎》《最后的晚餐》
   - **米开朗基罗**:《大卫》《创世纪》
   - **拉斐尔**:《雅典学院》

3. **科学思想**:
   - 哥白尼日心说（1543年）
   - 伽利略天文观测，支持日心说
   - 布鲁诺被烧死（1600年）

#### 北方文艺复兴
1. **伊拉斯谟**:《愚人颂》，批判教会
2. **莎士比亚**（英国）: 戏剧大师
3. **丢勒**（德国）: 版画艺术

### 1.2 宗教改革（16世纪）
#### 马丁·路德改革（1517年）
1. **九十五条论纲**: 反对赎罪券
2. **核心思想**:
   - "因信称义"
   - 圣经最高权威
   - 信徒皆祭司

3. **影响**: 路德宗建立，德国农民战争

#### 其他改革
1. **加尔文**（瑞士）:
   - 《基督教原理》
   - 预定论
   - 日内瓦神权共和国

2. **英国国教**:
   - 亨利八世与罗马教廷决裂（1534年）
   - 建立英国国教会

#### 反宗教改革
1. **耶稣会成立**（1540年）: 罗耀拉创立
2. **特兰托会议**（1545-1563年）: 整顿教会
3. **宗教裁判所**: 镇压异端

### 1.3 大航海时代（15-17世纪）
#### 葡萄牙探险
1. **亨利王子**: 设立航海学校
2. **迪亚士**: 绕过好望角（1488年）
3. **达·伽马**: 到达印度（1498年）

#### 西班牙探险
1. **哥伦布**: 发现美洲（1492年）
2. **麦哲伦**: 环球航行（1519-1522年）

#### 影响
1. **全球联系建立**
2. **殖民主义开始**
3. **物种大交换**
4. **商业革命**

## 二、科学革命与启蒙运动

### 2.1 科学革命（16-18世纪）
#### 物理学突破
1. **牛顿**:
   - 《自然哲学的数学原理》（1687年）
   - 万有引力定律
   - 三大运动定律
   - 微积分（与莱布尼茨独立发明）

2. **其他科学家**:
   - 开普勒行星运动定律
   - 波义耳化学研究
   - 哈维血液循环发现

#### 科学方法确立
1. **培根**: 归纳法，《新工具》
2. **笛卡尔**: 演绎法，"我思故我在"
3. **科学社团**: 皇家学会（1660年），法兰西科学院（1666年）

### 2.2 启蒙运动（17-18世纪）
#### 法国启蒙思想家
1. **伏尔泰**:
   - 批判专制和教会
   - 主张言论自由
   - "我不同意你的观点，但誓死捍卫你说话的权利"

2. **孟德斯鸠**:
   - 《论法的精神》
   - 三权分立学说
   - 影响美国宪法

3. **卢梭**:
   - 《社会契约论》
   - "人生而自由，却无往不在枷锁中"
   - 人民主权思想

4. **狄德罗**: 主编《百科全书》

#### 其他地区
1. **英国**: 洛克（天赋人权），休谟（经验主义）
2. **德国**: 康德（"敢于认识"）
3. **美国**: 富兰克林，杰斐逊

#### 影响
1. 为资产阶级革命提供思想武器
2. 推动社会政治改革
3. 促进科学文化发展

## 三、资产阶级革命时代

### 3.1 英国革命
#### 背景
1. 斯图亚特王朝专制
2. 清教徒受迫害
3. 议会与国王矛盾

#### 过程
1. **内战**（1642-1649年）:
   - 议会军vs王党军
   - 克伦威尔领导
   - 查理一世被处决（1649年）

2. **共和国时期**（1649-1660年）:
   - 克伦威尔护国公统治
   - 军事独裁

3. **光荣革命**（1688年）:
   - 威廉三世和玛丽二世即位
   - 《权利法案》（1689年）: 限制王权，议会主权
   - 君主立宪制确立

### 3.2 美国独立战争
#### 背景
1. 北美殖民地经济发展
2. 英国加重税收（印花税条例等）
3. "无代表不纳税"原则

#### 过程
1. **波士顿倾茶事件**（1773年）
2. **大陆会议召开**（1774年）
3. **莱克星顿枪声**（1775年）: 战争开始
4. **《独立宣言》**（1776年7月4日）: 杰斐逊起草
5. **萨拉托加大捷**（1777年）: 转折点
6. **约克镇战役**（1781年）: 英军投降
7. **巴黎和约**（1783年）: 英国承认美国独立

#### 制度建设
1. **《联邦条例》**（1781年）: 初期宪法
2. **1787年宪法**:
   - 三权分立
   - 联邦制
   - 第一部成文宪法
3. **《权利法案》**（1791年）: 前十项修正案

### 3.3 法国大革命
#### 背景
1. 三级会议制度不平等
2. 财政危机
3. 启蒙思想传播

#### 过程
1. **三级会议召开**（1789年5月）
2. **国民议会成立**（1789年6月）
3. **攻占巴士底狱**（1789年7月14日）: 革命开始
4. **《人权宣言》**（1789年8月）: 自由平等原则
5. **君主立宪时期**（1789-1792年）
6. **第一共和国**（1792-1804年）
7. **雅各宾派专政**（1793-1794年）: 罗伯斯庇尔，恐怖统治
8. **热月政变**（1794年7月）: 结束恐怖统治
9. **督政府时期**（1795-1799年）
10. **雾月政变**（1799年）: 拿破仑上台

#### 拿破仑时代
1. **《拿破仑法典》**（1804年）: 巩固革命成果
2. **军事扩张**: 几乎征服整个欧洲
3. **滑铁卢战役**（1815年）: 最终失败
4. **维也纳会议**（1815年）: 欧洲秩序重建

## 四、工业革命

### 4.1 第一次工业革命（18世纪60年代-19世纪中期）
#### 英国首先发生条件
1. 资产阶级统治确立
2. 圈地运动提供劳动力
3. 殖民扩张积累资本
4. 市场需求扩大
5. 科技知识积累

#### 主要发明
1. **纺织机械**:
   - 珍妮纺纱机（哈格里夫斯，1765年）
   - 水力纺纱机（阿克莱特，1769年）
   - 骡机（克朗普顿，1779年）
   - 动力织布机（卡特莱特，1785年）

2. **动力革命**:
   - 瓦特改良蒸汽机（1769年）
   - 蒸汽机车（斯蒂芬森，1814年）
   - 蒸汽轮船（富尔顿，1807年）

3. **冶金采矿**:
   - 焦炭炼铁（达比）
   - 搅炼法（科特）

#### 影响
1. **生产力飞跃**
2. **城市化加速**
3. **社会结构变化**: 工业资产阶级和无产阶级形成
4. **世界市场初步形成**

### 4.2 第二次工业革命（19世纪70年代-20世纪初）
#### 新特点
1. 科学理论与技术结合更紧密
2. 几乎同时在多国发生
3. 以电力和内燃机为标志

#### 主要成就
1. **电力应用**:
   - 发电机（西门子，1866年）
   - 电动机
   - 电灯（爱迪生，1879年）
   - 电报、电话、无线电

2. **内燃机和交通工具**:
   - 内燃机（奥托，1876年）
   - 汽车（本茨，1885年；戴姆勒，1886年）
   - 飞机（莱特兄弟，1903年）

3. **化学工业**:
   - 人造染料
   - 化肥
   - 炸药（诺贝尔）

4. **钢铁工业**:
   - 贝塞麦转炉炼钢法
   - 平炉炼钢法

#### 影响
1. **生产力进一步提高**
2. **垄断组织出现**
3. **世界市场最终形成**
4. **列强重新瓜分世界**

## 总结

世界近代史（约1500年-1900年）是人类社会从传统向现代转型的关键时期：

### 主要特征
1. **思想解放**: 文艺复兴、宗教改革、启蒙运动
2. **政治变革**: 资产阶级革命，民主制度建立
3. **经济革命**: 工业革命，资本主义发展
4. **全球联系**: 大航海时代，世界市场形成

### 历史意义
1. **现代世界奠基**: 政治、经济、思想基础奠定
2. **西方崛起**: 欧洲成为世界中心
3. **全球格局**: 殖民体系建立，东西方差距拉大
4. **社会转型**: 从农业社会向工业社会转变

世界近代史的发展为20世纪的现代世界准备了条件。
