# 世界历史时间线

## 🌍 概述
本时间线记录了从人类文明起源到21世纪的重要历史事件，为OpenClaw提供时空感知的基础。

## 📅 时间线结构

### 1. 史前时期 (公元前10000年以前)
**关键特征**: 人类起源、石器时代、农业革命

#### 重要事件：
- **约700万年前**: 人类与黑猩猩分化
- **约250万年前**: 旧石器时代开始，最早的石器工具
- **约1万年前**: 新石器时代，农业革命开始

### 2. 古代文明 (公元前4000年 - 公元500年)
**关键特征**: 大河文明、帝国兴起、文字发明

#### 主要文明时间线：
- **公元前3500年**: 苏美尔文明（美索不达米亚）
- **公元前3100年**: 古埃及统一
- **公元前2500年**: 印度河流域文明
- **公元前2070年**: 中国夏朝建立
- **公元前8世纪**: 古希腊城邦兴起
- **公元前6世纪**: 波斯帝国建立
- **公元前3世纪**: 孔雀王朝（印度）、秦朝（中国）
- **公元前27年**: 罗马帝国建立
- **公元1世纪**: 基督教诞生
- **公元4-5世纪**: 罗马帝国分裂与衰落

### 3. 中世纪 (公元500年 - 1500年)
**关键特征**: 封建制度、宗教影响、跨文化交流

#### 区域发展：
- **欧洲**: 查理曼帝国、十字军东征、黑死病
- **伊斯兰世界**: 伊斯兰教扩张、阿拉伯黄金时代
- **中国**: 唐宋盛世、元朝蒙古帝国
- **美洲**: 玛雅、阿兹特克、印加文明

### 4. 近代早期 (1500年 - 1800年)
**关键特征**: 地理大发现、文艺复兴、科学革命

#### 关键转折点：
- **1492年**: 哥伦布发现美洲
- **1517年**: 宗教改革开始
- **16-17世纪**: 科学革命（哥白尼、伽利略、牛顿）
- **1644年**: 清朝建立（中国）
- **18世纪**: 启蒙运动、工业革命开始

### 5. 近代晚期 (1800年 - 1945年)
**关键特征**: 工业革命、民族国家、世界大战

#### 重大事件：
- **19世纪**: 工业革命扩展、殖民主义高峰
- **1914-1918年**: 第一次世界大战
- **1917年**: 俄国十月革命
- **1929年**: 大萧条开始
- **1939-1945年**: 第二次世界大战

### 6. 当代世界 (1945年 - 至今)
**关键特征**: 冷战、全球化、数字革命

#### 时代划分：
- **1945-1991年**: 冷战时期
- **1947年**: 印度独立、印巴分治
- **1949年**: 中华人民共和国成立
- **1969年**: 互联网前身ARPANET建立
- **1989年**: 柏林墙倒塌
- **1991年**: 苏联解体
- **21世纪**: 全球化深化、气候变化、人工智能发展
