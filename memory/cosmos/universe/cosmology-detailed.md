## 🔭 宇宙基本理论

### 大爆炸理论
1. **证据支持**:
   - 宇宙微波背景辐射（2.725K）
   - 哈勃定律（星系红移）
   - 轻元素丰度（氢75%、氦25%）
   - 宇宙大尺度结构

2. **关键概念**:
   - 宇宙膨胀: 空间本身在膨胀
   - 宇宙学红移: 光波被拉长
   - 宇宙背景辐射: 大爆炸的余晖

### 暴胀理论
- **时间**: 大爆炸后10⁻³⁶到10⁻³²秒
- **膨胀倍数**: 至少10²⁶倍
- **解释问题**:
  - 平坦性问题（宇宙为何如此平坦）
  - 视界问题（宇宙为何如此均匀）
  - 磁单极问题（为何未见磁单极）

### 暗物质与暗能量
1. **暗物质** (27%宇宙质能):
   - 不发光，不吸收光，只通过引力作用
   - 证据: 星系旋转曲线、引力透镜、宇宙大尺度结构
   - 候选粒子: WIMPs、轴子、 sterile neutrino

2. **暗能量** (68%宇宙质能):
   - 导致宇宙加速膨胀
   - 性质: 负压强，均匀分布
   - 可能解释: 宇宙学常数、精质、修改引力理论

3. **普通物质** (5%宇宙质能):
   - 原子、分子、恒星、行星、生命

### 宇宙命运理论
1. **大撕裂** (Big Rip): 暗能量增强，撕裂所有结构
2. **热寂** (Heat Death): 宇宙持续膨胀冷却
3. **大挤压** (Big Crunch): 引力最终战胜膨胀
4. **大反弹** (Big Bounce): 循环宇宙模型
5. **多元宇宙**: 多个宇宙同时存在

## 🌟 恒星演化

### 恒星形成
1. **分子云**: 冷稠密气体云
2. **引力坍缩**: 云团在自身引力下收缩
3. **原恒星**: 中心温度压力升高
4. **主序星**: 氢核聚变开始

### 恒星分类
1. **按质量分类**:
   - 褐矮星 (<0.08 M☉): 未能点燃氢聚变
   - 红矮星 (0.08-0.5 M☉): 寿命极长，最常见
   - 类太阳恒星 (0.5-1.5 M☉): 太阳类型
   - 大质量恒星 (>1.5 M☉): 明亮，寿命短

2. **赫罗图分类**:
   - 主序带: 氢燃烧阶段
   - 巨星分支: 氢壳层燃烧
   - 水平分支: 氦核心燃烧
   - 渐近巨星分支: 双壳层燃烧

### 恒星死亡
1. **低质量恒星** (<8 M☉):
   - 红巨星 → 行星状星云 → 白矮星

2. **中等质量恒星** (8-25 M☉):
   - 超新星爆发 → 中子星

3. **大质量恒星** (>25 M☉):
   - 超新星爆发 → 黑洞

## 🪐 行星科学

### 行星形成理论
1. **星云假说**: 太阳系从旋转星云形成
2. **吸积过程**: 尘埃→星子→原行星→行星
3. **迁移理论**: 行星在形成过程中轨道迁移

### 行星分类
1. **类地行星**: 岩石组成，有固体表面
   - 水星、金星、地球、火星
2. **气态巨行星**: 主要由氢氦组成
   - 木星、土星
3. **冰巨行星**: 水、氨、甲烷冰组成
   - 天王星、海王星
4. **矮行星**: 未能清除轨道附近天体
   - 冥王星、谷神星、阋神星等

### 宜居带概念
1. **定义**: 行星表面可能存在液态水的轨道区域
2. **太阳系宜居带**: 约0.95-1.37 AU
3. **银河系宜居带**: 距星系中心适当距离
4. **宇宙宜居时代**: 恒星形成率适当的时期
