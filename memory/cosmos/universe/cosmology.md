# 宇宙学：宇宙的起源与演化

## 🌌 概述
宇宙学是研究宇宙整体性质、起源、演化和最终命运的科学。从大爆炸到可能的未来，本文件记录人类对宇宙的认知历程。

## 📏 宇宙尺度

### 1. 尺度层级
**从微观到宏观：**
- **量子尺度** (10⁻³⁵米): 普朗克长度
- **原子尺度** (10⁻¹⁰米): 原子大小
- **人类尺度** (1-10²米): 日常生活范围
- **行星尺度** (10⁶-10⁷米): 地球大小
- **恒星尺度** (10⁸-10¹²米): 太阳到恒星系统
- **星系尺度** (10²⁰-10²¹米): 银河系大小
- **宇宙尺度** (10²⁶米以上): 可观测宇宙

### 2. 时间尺度
- **宇宙年龄**: 约138亿年
- **太阳系年龄**: 约46亿年

## 💥 宇宙起源理论

### 大爆炸理论 (Big Bang Theory)
**核心观点**: 宇宙起源于一个极度高温高密度的奇点

#### 关键证据：
1. **宇宙微波背景辐射** (1965年发现): 大爆炸的"余晖"
2. **哈勃定律** (1929年): 星系红移，宇宙膨胀
3. **轻元素丰度**: 氢、氦、锂的比例符合预测
4. **宇宙大尺度结构**: 星系分布模式

#### 大爆炸时间线：
- **t=0**: 奇点，物理定律失效
- **10⁻³⁶秒**: 暴胀期开始，宇宙指数膨胀
- **1秒**: 核合成开始，形成轻元素
- **38万年**: 宇宙变得透明，光子解耦
- **2亿年**: 第一代恒星形成
- **10亿年**: 星系开始形成
- **90亿年**: 太阳系形成
- **138亿年**: 现在

### 暴胀理论 (Inflation Theory)
**解释问题**: 宇宙为何如此均匀、平坦

#### 关键特征：
- **指数膨胀**: 在10⁻³⁶到10⁻³²秒内膨胀10²⁶倍
- **量子涨落**: 形成宇宙结构种子
- **多重宇宙**: 可能产生平行宇宙

## 🔭 宇宙演化阶段

### 1. 普朗克时期 (0-10⁻⁴³秒)
- 量子引力主导
- 四力统一（假设）

### 2. 大统一时期 (10⁻⁴³-10⁻³⁶秒)
- 强力、弱力、电磁力统一
- 引力分离

### 3. 暴胀时期 (10⁻³⁶-10⁻³²秒)
- 宇宙指数膨胀
- 量子涨落放大

### 4. 再加热时期 (10⁻³²秒后)
- 暴胀能量转化为物质辐射
- 宇宙充满高温等离子体

### 5. 核合成时期 (1秒-3分钟)
- 质子中子结合形成轻元素
- 氢(75%)、氦(25%)、微量锂

### 6. 物质时期 (38万年后)
- 原子形成，宇宙透明
- 引力主导结构形成

## 🌠 宇宙组成

### 当前宇宙成分（WMAP数据）：
- **暗能量**: 68.3% - 加速宇宙膨胀
- **暗物质**: 26.8% - 引力作用但不发光
- **普通物质**: 4.9% - 恒星、行星、气体等
- **中微子**: 0.1% - 几乎无质量粒子
- **光子**: 0.01% - 宇宙微波背景

### 暗物质之谜：
- **证据**: 星系旋转曲线、引力透镜、宇宙大尺度结构
- **候选粒子**: WIMPs、轴子、 sterile neutrinos
- **探测方法**: 直接探测、间接探测、对撞机产生

### 暗能量之谜：
- **发现**: 1998年超新星观测显示宇宙加速膨胀
- **可能解释**: 宇宙常数、精质、修改引力理论
- **影响**: 决定宇宙最终命运

## 🌀 宇宙结构

### 层级结构：
1. **行星系统**: 恒星+行星+小天体
2. **恒星聚集**: 星团、星协
3. **星系**: 旋涡、椭圆、不规则
4. **星系群/团**: 数十到数千星系
5. **超星系团**: 星系团集合
6. **宇宙长城**: 星系纤维结构
7. **宇宙空洞**: 几乎没有星系的区域

### 可观测宇宙特征：
- **直径**: 约930亿光年
- **星系数量**: 约2万亿个
- **恒星数量**: 约10²⁴颗
- **年龄**: 约138亿年
- **平均温度**: 2.725K（微波背景）

## 🔮 宇宙未来

### 基于暗能量的三种可能命运：

#### 1. 大撕裂 (Big Rip)
- 暗能量密度随时间增加
- 宇宙加速膨胀无限加快
- 最终撕裂所有结构，包括原子

#### 2. 热寂 (Heat Death)
- 暗能量密度恒定或缓慢变化
- 宇宙持续膨胀但速率稳定
- 恒星燃料耗尽，黑洞蒸发
- 最终达到最大熵状态

#### 3. 大挤压 (Big Crunch)
- 暗能量密度减小，引力最终占优
- 宇宙膨胀停止并开始收缩
- 最终坍缩回奇点
- 可能引发新的大爆炸（振荡宇宙）
