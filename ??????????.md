## 🖱️ 鼠标原理深入分析

### 1. 鼠标硬件设计原理

#### 机械鼠标 (已淘汰):
```
机械结构:
    ├── 橡胶球 (滚动检测)
    ├── 水平/垂直滚轴 (机械编码器)
    ├── 光电传感器 (检测滚轴转动)
    └── 微动开关 (左右键点击)
        ↓
电信号输出:
    ├── X轴移动脉冲
    ├── Y轴移动脉冲
    ├── 左键点击信号
    └── 右键点击信号
```

#### 光电鼠标 (现代主流):
```
光学系统:
    ├── LED光源 (红光/蓝光/激光)
    ├── 透镜系统 (聚焦光线)
    ├── CMOS图像传感器 (拍摄表面图像)
    ├── DSP处理器 (图像对比分析)
    └── 微动开关 (按键点击)
        ↓
工作原理:
    1. LED照射桌面表面
    2. CMOS每秒拍摄数千张图像
    3. DSP对比连续图像计算位移
    4. 输出X/Y坐标变化
    5. 微动开关检测点击
```

#### 激光鼠标 (高端):
```
与光电鼠标类似，但:
    ├── 使用激光二极管 (更精确)
    ├── 更高DPI (12000+)
    ├── 表面适应性更强
    └── 功耗更低
```

### 2. 鼠标软件工作原理

#### Windows鼠标驱动栈:
```
硬件层:
    ├── USB HID设备 (人机接口设备)
    ├── USB控制器
    ├── 主板芯片组
        ↓
驱动层:
    ├── USB HID类驱动 (usbhid.sys)
    ├── 鼠标类驱动 (mouclass.sys)
    ├── 鼠标端口驱动 (mouhid.sys)
        ↓
系统层:
    ├── Windows输入子系统
    ├── 消息队列 (WM_MOUSEMOVE等)
    ├── 应用程序消息循环
        ↓
应用层:
    ├── 鼠标事件处理
    ├── 光标位置更新
    └── 点击事件响应
```

#### 鼠标数据流:
```
硬件 → 驱动 → 系统 → 应用
    ↓       ↓       ↓       ↓
USB数据 → HID报告 → 输入消息 → 事件处理
    ↓       ↓       ↓       ↓
坐标/点击 → 解析 → 分发 → 响应
```

## ⌨️ 键盘原理深入分析

### 1. 键盘硬件设计原理

#### 机械键盘:
```
机械结构:
    ├── 机械轴体 (Cherry MX等)
    ├── 弹簧 (提供回弹力)
    ├── 金属触点 (导通电路)
    ├── PCB电路板
    └── 控制器芯片
        ↓
工作原理:
    1. 按键按下 → 轴体压缩
    2. 金属触点接触 → 电路导通
    3. 控制器检测电路变化
    4. 生成扫描码
    5. 通过USB发送数据
```

#### 薄膜键盘:
```
薄膜结构:
    ├── 上层薄膜电路
    ├── 中层隔离层 (带孔)
    ├── 下层薄膜电路
    ├── 硅胶碗 (提供手感)
    └── 控制器芯片
        ↓
工作原理:
    1. 按键按下 → 硅胶碗压缩
    2. 上层薄膜接触下层薄膜
    3. 电路导通
    4. 控制器检测导通
    5. 生成扫描码发送
```

### 2. 键盘软件工作原理

#### Windows键盘驱动栈:
```
硬件层:
    ├── USB HID键盘设备
    ├── PS/2接口 (旧式)
    ├── 蓝牙键盘 (无线)
        ↓
驱动层:
    ├── USB HID类驱动 (usbhid.sys)
    ├── 键盘类驱动 (kbdclass.sys)
    ├── 键盘端口驱动 (i8042prt.sys或kbdhid.sys)
        ↓
系统层:
    ├── Windows输入子系统
    ├── 键盘布局管理器
    ├── 输入法编辑器 (IME)
    ├── 消息队列 (WM_KEYDOWN等)
        ↓
应用层:
    ├── 键盘事件处理
    ├── 字符输入
    └── 快捷键响应
```

#### 键盘数据流:
```
按键按下 → 扫描码 → 虚拟键码 → 字符码 → 应用程序
    ↓        ↓         ↓          ↓          ↓
物理按键 → 硬件编码 → 系统转换 → 字符映射 → 事件处理
```

```powershell
# 使用SendInput发送回车键
function Send-EnterKey {
    $inputs = New-Object INPUT[] 2

    $inputs[0].u.ki.wVk = 0x0D  # VK_RETURN

    $inputs[1].u.ki.time = 0

# 使用SendInput移动鼠标
function Move-MouseAbsolute {
    param([int]$x, [int]$y)
    
    $screenWidth = [System.Windows.Forms.Screen]::PrimaryScreen.Bounds.Width
    $screenHeight = [System.Windows.Forms.Screen]::PrimaryScreen.Bounds.Height
    
    # 转换为绝对坐标 (0-65535)
    $absX = [int]($x * 65535 / $screenWidth)
    $absY = [int]($y * 65535 / $screenHeight)
    
    $input = New-Object INPUT
    $input.type = [InputControl]::INPUT_MOUSE
    $input.u.mi.dx = $absX
    $input.u.mi.dy = $absY
    $input.u.mi.mouseData = 0
    $input.u.mi.dwFlags = [InputControl]::MOUSEEVENTF_MOVE -bor [InputControl]::MOUSEEVENTF_ABSOLUTE
    $input.u.mi.time = 0
    $input.u.mi.dwExtraInfo = [IntPtr]::Zero
    
    $inputs = @($input)
    [InputControl]::SendInput(1, $inputs, [System.Runtime.InteropServices.Marshal]::SizeOf([INPUT]))
}
```

### 3. 钩子(Hook)技术

#### 全局键盘钩子:
```powershell
# 设置全局键盘钩子
Add-Type @"
using System;
using System.Runtime.InteropServices;
using System.Windows.Forms;

public class KeyboardHook {
    private const int WH_KEYBOARD_LL = 13;
    private const int WM_KEYDOWN = 0x0100;
    private const int WM_KEYUP = 0x0101;
    private const int WM_SYSKEYDOWN = 0x0104;
    private const int WM_SYSKEYUP = 0x0105;
    
    private delegate IntPtr LowLevelKeyboardProc(int nCode, IntPtr wParam, IntPtr lParam);
    
    [DllImport("user32.dll", CharSet = CharSet.Auto, SetLastError = true)]
    private static extern IntPtr SetWindowsHookEx(int idHook, LowLevelKeyboardProc lpfn, IntPtr hMod, uint dwThreadId);
    
    [DllImport("user32.dll", CharSet = CharSet.Auto, SetLastError = true)]
    [return: MarshalAs(UnmanagedType.Bool)]
    private static extern bool UnhookWindowsHookEx(IntPtr hhk);
    
    [DllImport("user32.dll", CharSet = CharSet.Auto, SetLastError = true)]
    private static extern IntPtr CallNextHookEx(IntPtr hhk, int nCode, IntPtr wParam, IntPtr lParam);
    
    [DllImport("kernel32.dll", CharSet = CharSet.Auto, SetLastError = true)]
    private static extern IntPtr GetModuleHandle(string lpModuleName);
    
    private static IntPtr _hookID = IntPtr.Zero;
    private static LowLevelKeyboardProc _proc = HookCallback;
    
    public static void Start() {
        _hookID = SetHook(_proc);
    }
    
    public static void Stop() {
        UnhookWindowsHookEx(_hookID);
    }
    
    private static IntPtr SetHook(LowLevelKeyboardProc proc) {
        using (var curProcess = System.Diagnostics.Process.GetCurrentProcess())
        using (var curModule = curProcess.MainModule) {
            return SetWindowsHookEx(WH_KEYBOARD_LL, proc, GetModuleHandle(curModule.ModuleName), 0);
        }
    }
    
    private static IntPtr HookCallback(int nCode, IntPtr wParam, IntPtr lParam) {
        if (nCode >= 0) {
            int vkCode = Marshal.ReadInt32(lParam);
            
            // 这里可以处理按键事件
            // 例如：拦截特定按键，记录按键等
            
            // 示例：拦截F1键
            if (vkCode == (int)Keys.F1) {
                return (IntPtr)1;  // 阻止F1键传递
            }
        }
        
        return CallNextHookEx(_hookID, nCode, wParam, lParam);
    }
}
"@

# 启动键盘钩子
[KeyboardHook]::Start()

# 停止键盘钩子
[KeyboardHook]::Stop()
```

```powershell
    private static extern IntPtr CallNextHookEx(IntPtr hhk, int nCode, IntPtr wParam, IntPtr lParam);
    
    [DllImport("kernel32.dll", CharSet = CharSet.Auto, SetLastError = true)]

    public static void Start() {
        _hookID = SetHook(_proc);
    }
    
    public static void Stop() {
        UnhookWindowsHookEx(_hookID);
    }
    
    private static IntPtr SetHook(LowLevelMouseProc proc) {
        using (var curProcess = System.Diagnostics.Process.GetCurrentProcess())
        using (var curModule = curProcess.MainModule) {
            return SetWindowsHookEx(WH_MOUSE_LL, proc, GetModuleHandle(curModule.ModuleName), 0);
        }
    }
    
    private static IntPtr HookCallback(int nCode, IntPtr wParam, IntPtr lParam) {
        if (nCode >= 0) {
            MSLLHOOKSTRUCT hookStruct = (MSLLHOOKSTRUCT)Marshal.PtrToStructure(lParam, typeof(MSLLHOOKSTRUCT));

            if ((int)wParam == WM_MOUSEMOVE) {
                Console.WriteLine($"鼠标移动到: ({hookStruct.pt.x}, {hookStruct.pt.y})");
            }
        }
        
        return CallNextHookEx(_hookID, nCode, wParam, lParam);
    }
}
"@
```

## 🛠️ 解决自主苏醒问题的技术方案

### 1. 问题根本原因分析

#### 自主苏醒失败的可能原因：
1. **❌ 输入事件级别不够低**：
   - SendKeys可能被系统过滤或忽略
   - 需要更底层的输入控制

2. **❌ 窗口焦点问题**：
   - 测试窗口可能失去焦点
   - 输入事件可能发送到错误窗口

3. **❌ 系统空闲检测机制**：
   - Windows有复杂的空闲检测逻辑
   - 简单的回车键可能不足以重置空闲计时器

4. **❌ 权限和安全限制**：
   - 某些输入API需要管理员权限
   - 安全软件可能拦截自动化输入

### 3. 窗口焦点管理方案

#### 确保窗口焦点：
```powershell
function Ensure-WindowFocus {
    param([string]$windowTitle)
    
    # 使用Windows API激活窗口
    Add-Type @"
using System;
using System.Runtime.InteropServices;
public class WindowControl {
    [DllImport("user32.dll")]
    public static extern bool SetForegroundWindow(IntPtr hWnd);
    
    [DllImport("user32.dll", SetLastError = true)]
    public static extern IntPtr FindWindow(string lpClassName, string lpWindowName);
    
    [DllImport("user32.dll")]
    public static extern bool ShowWindow(IntPtr hWnd, int nCmdShow);
    
    public const int SW_RESTORE = 9;
    public const int SW_SHOW = 5;
}
"@
    
    # 查找窗口
    $hwnd = [WindowControl]::FindWindow($null, $windowTitle)
    if ($hwnd -ne [IntPtr]::Zero) {
        # 恢复窗口
        [WindowControl]::ShowWindow($hwnd, [WindowControl]::SW_RESTORE)
        
        # 置顶窗口
        [WindowControl]::SetForegroundWindow($hwnd)
        
        Write-Host "✅ 窗口焦点已确保: $windowTitle" -ForegroundColor Green
        return $true
    } else {
        Write-Host "❌ 未找到窗口: $windowTitle" -ForegroundColor Red
        return $false
    }
}
```
