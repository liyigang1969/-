# 🦞 元灵邻居地图 - 部署说明

## 文件
- `nearby_map.html` — 地图页面源码
- `deploy_map.sh` — 一键部署脚本（放到服务器上执行）

## 部署
```bash
# 1. 把 deploy_map.sh 传到服务器
scp deploy_map.sh root@服务器IP:/root/

# 2. 在服务器上执行
bash deploy_map.sh

# 3. 访问
# http://服务器IP:6789
# 微信内直接打开即可
```

## 原理
- 使用高德地图 Web JS API v2.0
- 自动定位用户位置
- 显示附近的"元灵邻居"（演示数据）
- 信息窗口显示邻居的经验数+活跃状态
- 深色风格，移动端适配（微信里效果最好）
