你说的是："就是一个接管我的鼠标键盘的批处理的循环程序"

## 📝 自定义修改

### 修改等待时间：
```batch
:: 默认等待3秒
timeout /t 3 /nobreak >nul

:: 改为等待10秒
timeout /t 10 /nobreak >nul
```

### 添加自定义操作：
```batch
:: 在 :loop 标签后添加
echo 执行我的自定义操作...

:: 例如：创建时间戳文件
echo %date% %time% > timestamp.txt

:: 例如：播放提示音
echo 
```

### 修改显示信息：
```batch
:: 修改状态显示
echo [%time%] 我的程序运行中 - 第 %count% 次循环
```
